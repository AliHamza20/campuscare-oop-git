# CampusCare

A console-based campus maintenance complaint management system.

## How to Run
1. Open project in IntelliJ IDEA
2. Run Main.java
3. Use the menu to add persons and create tickets
## Notes
- Supports Maintenance and Cleaning ticket types
- Priority score calculated automatically
